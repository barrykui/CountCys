# CountCys
Count the number of Cys in a series of fasta formatted protein sequences  


# Manual

```shell
CountCys   : Count Cys in fasta file.
Version : 0.0.1
Usage   : java -jar countcys.jar -f test.fasta
 -f  <str>    Protein fasta file.

---
 -v           View output.
 -h           View help option.
```
